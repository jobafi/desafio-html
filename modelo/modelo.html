Desafio HTML básico

O desafio de HTMl básico do curso de Desenvolvimento Web Full Stack, consiste em criar um currículo profissional pessoal baseado em um modelo disponibilizado pelo curso, usando o que foi estudado até o momento.


Aqui, você encontra a página do curso.


Clique aqui e confira o modelo disponibilizado.


Foi solicitado usar:

  • Tags;
  • Títutlos e Parágrafos;
  • Links;
  • Quebra de Linha;
  • Imagens;
  • E inserção de linha horizontal.

Incrementações não solicitadas (adicionais):

  • Aplicação de propriedades CSS: width, height, display, font-family e outros;
  • Criação de class para aplicação de propriedades CSS;
  • Criação de página específica sobre o desafio (esta página que você está lendo);
  • Envio e disponibilização pública no GitHub.
